Scrollama Summary

Hi!

First disclaimer: the attached examples are the ones from the scrollama website. I just split them out and removed all internal references so that they'd operate as stand alones and the code could be more easily understood.

Okay, so (assuming I'm understanding correctly), scrollama is pretty straight forward: it is a listening device for scrolling. So you set a line on the page and then listen for when scrolled items cross that line.

When in development mode, this line can be set to visible -- which it is in all of the examples below.

The power of scrollama is listening to the relationahip between the scrolling items and this customizable line. I'm not sure if this is comprehensive, but you can know:

  • when scrolled items first overlap the line ("onStepEnter")
  • when scrolled items are no longer overlapping the line ("onStepExit")
  • the % that the scrolled item is overlapped by the line ("onStepProgress")

These examples are all dependent on interaction-objsever.js and two of them use stickyfill.js. D3 is also used in two of them but unlike the other two in the previous sentence, we could swap this out if you didn't want to use this.
